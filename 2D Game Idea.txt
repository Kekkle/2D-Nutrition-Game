2D Game Idea: 

I would like to create a 2D style game, similar to the simple Mario game where he jumps along and has to collect coins and avoid negative things. 

This 2D game will be based on Nutrition pricinples. For example, the first level will focus on "collect all the Carbs" Next level "Collect Protein" then "Collect Fats". The Concepts and learning woul dbe like:

-Learn about Carbs (L1), Proteins (L2) Fats (L3)
-Learn about Glucose (L4), amino acids, and lipids that make up each macronutritient.
-Identify foods based on glucose, amino acid, lipids. 
-Learn about energy needs. Carb = quick energy. Fats = Slower energy source. Protein, not really for energy, mostly structure. 
-Learn about low energy availability. Breakdown of tissue and fats for glucose substrate for energy. 
-Learn about ineffeciency of ultra processed foods like chicken nuggets and fries. How eating those things provides quick, short-term energy and doesn't fuel (or satiate for long). 
-Learn about bodies needs for energy. Brain has high energy demands. Same as immune system and digestive tract. How systems run less well when energy is low. 


So the idea is that the start of the game level would have a simple piece of info on the opening window to "teach" the basic concept for that level like "Carbs are found in such and such items, and provide such and such" then the user collects all carbs. The idea is that the instruction window is the only real learning moment, so it's simple but memorable. 

There would need to be consistent mods for the user to avoid or be wary of, but I'm not sure what those are yet. 

User skill should include running, jumping and walking and ducking. User gains points and coins as they go along which can unlock momentary new powers, like gliding down from a jump.
