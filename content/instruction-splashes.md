# Instruction Splash Copy — Tier 1 (Levels 1–4)

Each splash is the player's only "teaching moment." Max 2–3 sentences of fact, a clear mission, and one hazard/enemy preview. Design guidelines are in GAME_PLAN.md Section 6.

All copy has been checked against `.cursor/rules/content-guardrails.mdc`.

---

## Level 1: Carbohydrates

```
┌──────────────────────────────────────────────┐
│  LEVEL 1: CARBOHYDRATES                      │
│                                              │
│  [Icons: bread, rice, banana, potato]        │
│                                              │
│  "Carbs are your body's preferred            │
│   quick-energy fuel. They break down into    │
│   glucose, which powers your muscles and     │
│   your brain."                               │
│                                              │
│  YOUR MISSION: Collect all the carb-rich     │
│  foods! Some also contain fibre — grab       │
│  those for bonus points.                     │
│                                              │
│  WATCH OUT: Fatigue Phantoms drain your      │
│  energy if they catch you!                   │
│                                              │
│              [ START ]                       │
└──────────────────────────────────────────────┘
```

**Copy notes:**
- Icons should be instantly recognizable pixel-art food items.
- "Quick-energy fuel" establishes carbs' functional identity without ranking macronutrients.
- The fibre bonus hint foreshadows L4 without overloading the player.

## Level 2: Protein

```
┌──────────────────────────────────────────────┐
│  LEVEL 2: PROTEIN                            │
│                                              │
│  [Icons: egg, chicken leg, beans, fish]      │
│                                              │
│  "Protein is your body's building material.  │
│   Every tissue — muscle, skin, organs, even  │
│   your immune cells — is built and repaired  │
│   with protein."                             │
│                                              │
│  YOUR MISSION: Collect protein-rich foods!   │
│  Pair plant proteins together (like rice +   │
│  beans) for a combo bonus.                   │
│                                              │
│  WATCH OUT: Stress Sprites move in erratic   │
│  clusters — stay alert!                      │
│                                              │
│              [ START ]                       │
└──────────────────────────────────────────────┘
```

**Copy notes:**
- "Building material" is the core metaphor for protein — reinforced throughout L2.
- The combo mechanic is introduced here so the player knows to look for pairings.
- Stress Sprites are the new enemy introduced in L2 — erratic movement raises difficulty.

## Level 3: Fats

```
┌──────────────────────────────────────────────┐
│  LEVEL 3: FATS                               │
│                                              │
│  [Icons: avocado, salmon, nuts, olive oil]   │
│                                              │
│  "Fats are your body's slow-burn energy      │
│   source. They build every cell membrane     │
│   and help you absorb essential vitamins."   │
│                                              │
│  YOUR MISSION: Collect a variety of fats!    │
│  The more types you collect, the more        │
│  sustained your energy becomes.              │
│                                              │
│  WATCH OUT: Free Radicals bounce off walls   │
│  unpredictably — and they split when you     │
│  jump on them!                               │
│                                              │
│              [ START ]                       │
└──────────────────────────────────────────────┘
```

**Copy notes:**
- "Slow-burn" contrasts with L1's "quick-energy," building a natural carbs-vs-fats framework.
- "Variety" is the mission keyword — no fat type is singled out as bad.
- Free Radicals raise difficulty with their splitting mechanic (one becomes two).

## Level 4: Fibre

```
┌──────────────────────────────────────────────┐
│  LEVEL 4: FIBRE                              │
│                                              │
│  [Icons: broccoli, raspberries, oats,        │
│   lentils, popcorn]                          │
│                                              │
│  "Some carbs contain fibre (think fruits and │
│   veg). Fibre feeds the trillions of helpful │
│   bacteria in your gut. They keep your       │
│   digestion strong, support your immune      │
│   system, and even help your brain."         │
│                                              │
│  YOUR MISSION: Deliver diverse fibre to      │
│  the microbiome stations! Different          │
│  sources power up different stations.        │
│                                              │
│  WATCH OUT: Entropy Blobs expand to block    │
│  your path — move fast before they grow!     │
│                                              │
│              [ START ]                       │
└──────────────────────────────────────────────┘
```

**Copy notes:**
- "Trillions of helpful bacteria" is memorable and accurate.
- The mission introduces the microbiome station mechanic — the player needs to deliver varied fibre, not just volume.
- Entropy Blobs are the Tier 1 finale enemy — they pressure the player to move quickly and plan routes.
- "Most people don't get enough" was considered but cut to avoid any hint of dietary shaming. The emphasis is on diversity instead.
