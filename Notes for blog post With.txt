my ideas for the game: 
I would like to create a 2D style game based on Nutrition pricinples. For example, the first level will focus on "collect all the Carbs" Next level "Collect Protein" then "Collect Fats". Principles would be acquired simply through game play, not reading pr listening to information. For example, when the user jumps along and doesn't collect carbs, they lose energy, each time they eat a carb, energy gained. etc. 

So the idea is that the start of the game level would have a simple piece of info on the opening window to "teach" the basic concept for that level like "Carbs are found in such and such items, and provide such and such" then the user collects all carbs. The idea is that the instruction window is the only real learning moment, so it's simple but memorable.
